# 📸 MidnightScreenshot

Автоматическое создание скриншотов выбранной области в полночь (или в режиме тестирования). Кроссплатформенный инструмент для Windows, Linux и macOS.

## 🌟 Возможности

- ✅ Интерактивный выбор области скриншота через GUI
- ✅ Автоматическое сохранение скриншотов в полночь
- ✅ Автозапуск при загрузке ОС
- ✅ Кроссплатформенность (Windows, Linux, macOS)
- ✅ Проверка совместимости (Wayland, DPI масштабирование)
- ✅ Обработка ошибок и детальное логирование

## 📋 Требования

- Python 3.7+
- pip или другой менеджер пакетов

## 🚀 Установка

```bash
# Клонировать репозиторий
git clone https://github.com/ВАШ_НИК/MidnightScreenshot.git
cd MidnightScreenshot

# Установить зависимости
pip install -r requirements.txt
```

## 💻 Использование

### Тестовый режим (3 сек вместо полночи)

```bash
python ZeroShot.py
```

1. Нажмите на **левый верхний угол** нужной области
2. Нажмите на **правый нижний угол** нужной области
3. Скриншот будет сохранен в папку `PHOTOS/` через 3 секунды (или в полночь, если выключить тестовый режим)

### Боевой режим (полночь)

Отредактируйте в `ZeroShot.py`:
```python
TEST_MODE = False  # Вместо True
```

## ⚙️ Конфигурация

После первого запуска создается файл `config.txt` с координатами. Вы можете отредактировать его вручную:

```
x1,y1
x2,y2
```

## 🔧 Автозапуск

При первом запуске скрипт автоматически регистрируется для запуска при загрузке:

- **Windows**: Добавляется в реестр (HKCU\Software\Microsoft\Windows\CurrentVersion\Run)
- **Linux**: Создается файл в `~/.config/autostart/`
- **macOS**: Создается LaunchAgent в `~/Library/LaunchAgents/`

## 🐛 Решение проблем

### "Wayland обнаружен"
Переключитесь на X11 для лучшей совместимости:
```bash
export QT_QPA_PLATFORM=xcb
```

### Скриншоты сохраняются весь экран, а не область
- Убедитесь, что DPI масштабирование отключено на Windows
- На Linux попробуйте X11 вместо Wayland

### Скрипт не запускается автоматически
- Проверьте права доступа: `chmod +x ZeroShot.py`
- Убедитесь, что Python находится в PATH
